# 마크다운 사용법

1. '#'을 이용해서 제목, 부제목들을 입력할 수 있다.
   - ex1) # Heading 하면 제목
   - ex2) ## Heading 하면 부제목
2. '\_\_\_' (언더바3개)를 이용해서 수평선을 추가할 수 있다.
3. 다양한 텍스트 속성
   - '\*' 을 이용해서 원하는 문자를 감싸면 기울리기 가능
   - '\*\*' => bold체
   - '~~' => stripe체
4. 원하는 문장을 인용체구로 집어 넣고 싶을 때는 '>' 사용

   > ex 인용구안으로 넣은 예시
